# Ionic Jade Scaffold

#### Installation

* Works with node 4 only

> npm install -g cordova ionic

> npm install

> gulp install

> ionic state restore

#### Development

> ionic serve

### Build
> ionic platform add <platform>

> ionic build

#### Configure

To change application's name, change "starter" in the following files:
* package.json
* ionic.project
* config.xml
* bower.json
* jade/index.jade
* www/js/app.js
* www/js/routes.js
